# Project Overview

This repository contains my completed Data Analytics Excel project, designed to strengthen practical spreadsheet, data cleaning, and analytical problem-solving skills through business-focused exercises.

---

# Project Objectives

The primary objective of this project was to develop and apply essential Microsoft Excel skills in solving practical business data challenges. Throughout the project, the focus was on cleaning and organizing raw datasets, improving data quality by handling incomplete records, and structuring information for easier analysis.

This project also aimed to strengthen analytical thinking by using Excel functions and formulas to answer business questions, and identify patterns. Key objectives included applying lookup functions, performing calculations such as margin and commission analysis, improving reporting presentation, and building confidence in spreadsheet-based decision-making.

By completing these tasks, the project supports foundational growth in data analytics, reporting, and business intelligence.

# Excel Formulas Used

| Purpose | Formula |
|--------|---------|
| Commission | `=H2*$L$1` |
| Margin | `=H2-I2` |
| Find Rock | `=FIND($N$3,B2)` |
| Album Lookup | `=VLOOKUP("Top Top",B:J,2,FALSE)` |
| Genre Lookup | `=VLOOKUP("Top Top",B:J,5,FALSE)` |
| Count Support Reps | `=COUNTIF($M:$M,O2)` |
